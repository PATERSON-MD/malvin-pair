PATERSON-MD

Solution avancée de bot WhatsApp avec sessions sécurisées, optimisée pour le cloud

Connexion QR Code

Connectez votre compte WhatsApp en scannant un QR code sécurisé. Simple, rapide et ne nécessite pas de partager votre numéro.

Ouvrir le Générateur QR
terminal@paterson-md:~
$ qr-generator --secure
> QR session initialized
> Waiting for device pairing...

Connexion Pairing Code

Connectez-vous avec un code d'appairage unique. Entrez votre numéro, recevez un code et connectez votre compte WhatsApp en toute sécurité.

Générer un Code
terminal@paterson-md:~
$ pair-session --number +509XXXXXXX
> Security code generated
> Code valid for 10 minutes...

Déploiement & Personnalisation

Obtenez votre propre instance de PATERSON-MD. Forkez le dépôt GitHub, personnalisez et déployez sur Render en quelques minutes.

Fork sur GitHub Déployer sur Render
terminal@paterson-md:~
$ git clone https://github.com/PATERSON-MD/PATERSON-MD.git
$ cd PATERSON-MD
$ npm install
$ npm start
